# Funcionalidades do Sistema

## Visão geral

Sistema SaaS de gestão para gráfica e vendas com operação interna, catálogo público, pedidos online, PDV, financeiro, estoque, produção e administração multiempresa.

## Autenticação e acesso

- Login de equipe por e-mail e senha
- Recuperação de senha
- Alteração obrigatória de senha quando configurada
- Onboarding inicial da empresa
- Perfis de acesso por cargo
- Controle de permissões por módulo
- Separação entre acesso de loja e super admin
- Login e portal do cliente para acompanhar pedidos
